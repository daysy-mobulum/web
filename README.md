# Daysy

**Event Countdown Tracker** — Know exactly how many days until your next important event.

Live at: [daysy.mobulum.com](https://daysy.mobulum.com)

## What is Daysy?

Daysy is a free, privacy-first web application that helps you track upcoming events, holidays, and important dates with countdown timers. It comes pre-loaded with national holidays for 50 countries.

## Features

- Countdown timers showing days until each event
- Pre-loaded national holidays for 50+ countries
- Recurring events (weekly, monthly, yearly)
- Tag-based organization and filtering
- Multi-language UI (15 languages)
- Dark/Light/System theme
- Fully offline — all data stored in your browser
- Data export/import (JSON)
- No accounts, no tracking, no ads

## Privacy

Daysy does not collect any data. There are no analytics, no cookies, no server calls. Everything is stored locally in your browser's localStorage. Your data never leaves your device.

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- React Router (history mode)
- react-i18next
- Vitest + Testing Library
- GitHub Pages
